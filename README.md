# TennisComp Registration System

A single-page web application for tennis players to register and track their skill level, find matches, and compete with others.

## Description

This project is a tennis player registration form built with HTML, CSS, and JavaScript. It demonstrates DOM manipulation, form validation, and event-driven programming. Users can create an account by providing personal information, selecting court preferences, and indicating their skill level.

### Dependencies

* Modern web browser (Chrome, Firefox, Safari, Edge)
* No additional libraries or frameworks required
* Works on Windows, macOS, and Linux

### Installing

* Download or clone the repository from GitHub
* No modifications needed - all files are ready to use
* Ensure the folder structure is maintained:
```
  project/
  ├── html/
  │   └── jssba.html
  ├── css/
  │   └── jssba.css
  └── js/
      └── jssba.js
```

### Executing Program

* Open the `jssba.html` file in your web browser
* Fill out the registration form:
  1. Enter your first and last name
  2. Provide a valid email address
  3. Create a password (8-20 characters)
  4. Confirm your password
  5. Select your gender
  6. Enter your date of birth (must be between 13 and 100)
  7. Choose at least one court preference
  8. Select your skill level
* Click the "Register" button to submit

## Features

* Real-time form validation
* Password strength requirements (8-20 characters)
* Password matching confirmation
* Email format validation
* Required field checking
* Court preference validation (at least one must be selected)
* Dynamic success messaging using HTML templates
* Responsive design with earth-tone color scheme
